## Key Conventions

- Python dependencies in requirements.txt — install with `pip install -r requirements.txt`
- All bash scripts use `set -e` and source `.claude/skills/common/checks.sh` for prereq checks
- Python scripts import helpers from `.claude/skills/common/python_checks.py`
- Scripts return ≤5 lines to stdout; verbose output goes to `/tmp/<skill>-<action>.log`

## Skills Available

The following Skills are available in `.claude/skills/`. Trigger them with plain-English prompts:

  - `agents-md-gen`: Generate or update AGENTS.md for any repository. Inspects directory structure,
  - `docusaurus-deploy`: Build and deploy a Docusaurus documentation site to Kubernetes. Runs npm build,
  - `fastapi-dapr-agent`: Scaffold a FastAPI microservice with Dapr sidecar, Dockerfile, and K8s manifests
  - `kafka-k8s-setup`: Deploy Apache Kafka on Kubernetes using Bitnami Helm chart (chart 32.4.3, Kafka 
  - `mcp-code-execution`: Generate a Python MCP client script that wraps MCP server calls for token effici
  - `nextjs-k8s-deploy`: Build a Next.js application Docker image and deploy it to Kubernetes on Minikube
  - `postgres-k8s-setup`: Deploy PostgreSQL on Kubernetes (Bitnami chart 18.5.1, PostgreSQL 16) and run

Run `bash .claude/skills/common/preflight.sh` to verify all prerequisites.

## Getting Started

1. Ensure Minikube is running: `minikube start --cpus=4 --memory=8192`
2. Run preflight: `bash .claude/skills/common/preflight.sh`
3. Start Claude Code or Goose from this directory
4. Use plain-English prompts — skills are auto-triggered
